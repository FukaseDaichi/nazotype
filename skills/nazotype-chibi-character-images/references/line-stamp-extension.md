# LINE Stamp Extension

## Purpose

Keep the base chibi asset stable enough that later LINE sticker work can branch from it without redesigning the character.

## Preserve Now

Lock these into the approved base chibi:

- face identity
- hairstyle silhouette
- outfit silhouette
- core props
- main palette
- character attitude

These are the anchors that later expressions and gestures should inherit.

## Recommended Expansion Path

1. Approve one strong base chibi per type.
2. Treat that approved chibi as the main identity anchor for later sticker prompts.
3. Generate expression and gesture variants from that approved base, not from the old standard art alone.
4. Add text only in the dedicated LINE workflow.

## Identity Consistency Rule

When generating future sticker variants, prefer reference-image workflows from the approved chibi base over prompt-only regeneration whenever possible.

## Non-Goal

This skill does not place text, build final LINE canvases, or handle sticker packaging.
