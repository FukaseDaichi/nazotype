# LINE Stamp Extension

## Purpose

Keep the current character generation workflow compatible with future LINE sticker production.

## Preserve Now

Keep these stable in the base character assets:

- hairstyle
- face shape
- outfit silhouette
- main props
- recognizable color palette

These are the anchors that make later expression and pose variations feel like the same character.

## Future Additions

Add these later rather than now:

- expression sets
- gesture and pose sets
- text bubble or lettering overlays
- sticker-safe padding rules
- export presets for sticker delivery

## Recommended Expansion Path

1. Generate and lock a good `standard` base per type.
2. Generate a matching `chibi` base per type.
3. Create expression variants from the chosen base identity.
4. Add per-stamp text overlays only after the character identity is stable.

## Identity Consistency Rule

When generating future sticker variants, prefer image-to-image or reference-image workflows over pure text-only regeneration whenever possible.
